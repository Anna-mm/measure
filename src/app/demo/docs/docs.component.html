How to use

Install

$ npm install --save measure-ui

Usage

import {UIButtonModule} from 'measure-ui';

@NgModule({
    // ...
    imports: [
    // ...
    UIButtonModule
    ]
})
export class AppModule {}

Styles

You can use the default theme, just import the following css file. 
@import "../node_modules/measure-ui/asset/style/measure.min.css";
Or You can customize the theme, when you use less. 
/* You can add global styles to this file, and also import other style files */
@import "../node_modules/measure-ui/asset/style/component.less";

// your theme variables
@brand-primary: #f0f;
The available variables can be find in node_modules/measure-ui/asset/style/variable.less.
