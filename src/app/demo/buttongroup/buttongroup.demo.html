button group

radio button group(default)

<ui-buttongroup [datasource]="data"></ui-buttongroup>

radio button group with value

<ui-buttongroup [datasource]="data" [value]="3"></ui-buttongroup>

checkbox button group

<ui-buttongroup [datasource]="data" type="checkbox"></ui-buttongroup>

checkbox button group with value

<ui-buttongroup [datasource]="data" type="checkbox" [value]="[1, 3]"></ui-buttongroup>

use within forms

When use with forms, you can use ngModel instead of value.

form value is: {{frm.value | json}} 
<form #frm="ngForm">
    <ui-buttongroup name="btngroup" [datasource]="data" type="checkbox" [ngModel]="[1, 3]"></ui-buttongroup>
</form>

<div>
    form value is: {{frm.value | json}}
</div>
